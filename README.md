# Chatbot

A simple chatbot project built with Python.

## Features

- Interactive command-line chatbot
- Easy to extend with new responses
- Beginner-friendly codebase

## Requirements

- Python 3.x

## Installation

```bash
git clone https://github.com/yourusername/chatbot.git
cd chatbot
```

## Usage

```bash
python chatbot.py
```
